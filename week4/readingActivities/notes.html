Chp 8: Forms

Forms Reading Activity

Forms

  • Traditionally: info on form submitted and sent to a server where PHP or Ruby would process it.
  • Now: can process info in a form on the front end before its sent to the server using JavaScript
  • Simple form example:

 

    <form name='search' action='/search'>

        <input name='searchInput'>

        <button type='submit'>Search</button>

    </form>

 

Accessing Form Elements

  • To receive info submitted in form use this:
    • const form = document.forms[0];
    • Which is the same as const form
      • getElementsByTagname('form')[0];
    • You can also use the name of the name of the attribute in the form:
      • const form = document.forms['search'];
    • The object form used above also has a method called elements that returns a collection of all the elements in the form. This form has two elements in it and the values of those are set to be put into variables:
      • const [input,button] = form.elements;

Form Properties and Methods

  • Ways to submit info on a form:
    • <button type='submit'>Submit</button>
    • <input type='submit' value='Submit'>
    • <input type='image' src='button.png'>
  • Reset form controls back to initial values
    • <button type='reset'>Reset</button>
  • Can send info to a different URL to be processed on the server using action property
    • action = '/an/other.url'

Form Events

  • focus event: on an <input> element the focus is when the cursor is placed inside the element.
    • This would be the JS for that and it creates a pop up alert with ‘focused’ in it:
      • const input = form.elements.searchInput;
      • addEventListener('focus', () => alert('focused'), false);
  • blur event
    • This would blur the form. Add this to JS:
      • addEventListener('blur', () => alert('blurred'), false);
  • change event
    • when user moves the focus away from the form after changing it. Add this to JS
      • addEventListener('change', () => alert('changed'), false);
  • submit event
    • usually this would send content to the server to be processed, but JS can intercept the form before its sent by adding a submit event listener. JS for that
      • const form = document.forms['search'];
      • addEventListener ('submit', search, false);
      • function search() {
      • alert(' Form Submitted');
      • }

Chp 12 Object-Oriented Programming in JavaScript

3 main concepts

  • Encapsulation
    • Inner workings are kept hidden inside the object and only the essential functionalities are exposed to the end user. All programming logic is inside the object
  • Polymorphism
    • The same process can be used for different objects. Various objects can share the same method, but also have the ability to override shared methods with a more specific implementation.
  • Inheritance
    • Taking the features of one object then adding some new features. We can take an object that already exists and inherit all its properties and methods. We can then improve on its functionality by adding new properties and methods.

Classes

  • Java is a class based langues. Classes define a blueprint for an object. Objects inherit all the properties and methods of the class.
  • JavaScript has classes as of ES6.
