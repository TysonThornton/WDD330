Local Storage Notes

  • Local Storage is a way to store values in the console that can be used
  • Code Example from Bro Thayne

    • if (localStorage.clickcount) { localStorage.clickcount = Number(localStorage.clickcount) + 1; } else { localStorage.clickcount = 1; } document.getElementById("result").innerHTML = "You have clicked the button " + localstorage.clickcount + " time(s).";

  • Two ways to SET local storage in JavaScript:

    • localStorage.setItem("firstName", "Tyson");
    OR
    localStorage.lastName = 'Thornton';

  • Two ways to GET local storage in JavaScript:

    • let firstName = localStorage.getItem('firstName');
    OR
    let lastName = localStorage.lastName;

Master Mobile UX Video Notes

  • "Whatever you are doing, do mobile first."
  • Start small and scale up - focus on content and functionality
  • Design
    • Speed Matters: reduce image usage throughout sites/apps, SVG/Webfonts are lower bandwidth, Optimize and Minify CSS and JS files, GZIP files to eliminate unnecessary data.
    • Think about what how users use technology -- menu should be at the bottom of mobile.
  • Forms
    • Put labels inside the box, make the boxes bigger with mroe padding, once a field is filled out put the label above the input.
    • Suggestions: provide optimal keyboard, use a a ref link on phone numbers, makingin CC entry better.
