Chp 9 The Window Object

  • Window object: every JavaScript environment has a global object. Any global scope variable are properties of this object automatically and any functions are methods of it

Browser Object Model

  • Is a collection of properties and methods that contain info about the browser and computer screen. It can be used to find dimensions of user’s screen, which pages have been visited before, and can create pop up windows.

Making a variable global.

  • It has to be set as a property of the window object
  • Example
    • x = 6; // global variable created
    • << 6
    • x // same variable can be accessed as a property of the window object
    • << 6
    • // both variables are exactly the same
    • x === x;
    • << true

Dialogs

  • Alert() – pause execution of program to display a message in dialog box
  • confirm() – stop execution of program to display confirmation dialog and gives options of OK or Cancel. Returns a Boolean value
  • prompt() – stop execution of program and displays dialog box with an input field to all user to input text
  • All browsers support them as methods of window object
    • alert('Hello');
    • confirm('Do you wish to continue?');

Location

  • location is an object that has info about the URL of current page.
    • location.href -- returns full URL as a string
    • location.host -- returns the domain of current URL
    • location.hostname
    • location.port
    • location.search -- returns empty string if no query string parameters
  • location also has methods
    • reload() method can be used to force a reload of the current page. If it’s given a parameter of true , it will force the browser to reload the page from the server, instead of using a cached page.
    • assign() method can be used to load another resource from a URL provided as a parameter
    • replace() method is almost the same as the assign() method, except the current page will not be stored in the session history, so the user will be unable to navigate back to it using the back button.
    • The toString() method returns a string containing the whole URL

Other Properties

  • history – access info about previously visited pages
  • open() – a new window can be opened.
    • const popup = window.open('https://sitepoint.com','
    • SitePoint','width=400,height=400,resizable=yes');
  • Screen info
    • screen.height
    • screen.width

Document Object

  • Write to the page

Cookies

  • Create: document.cookie = 'name=Superman';
  • Change: document.cookie = 'name=Batman'
  • Reading:
    • cookie:
    • << "name=Batman; hero=true; city=Gotham"
  • Delete: set expire time, or if it’s a session cookie it will expire when window is closed
    • cookie = 'name=Batman; expires=Thu, 01 Jan 1970 00:00:01 GMT';
