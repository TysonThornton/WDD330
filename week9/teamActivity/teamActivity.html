Instructions

Core Requirements

  1. Download the starter code and add the Javascript necessary to make the appropriate sound file play when the corresponding key is pressed.

  2. Make it quick.

    You may have noticed that if you press a key rapidly the sound file does not immediately re-play. Modify your code to make the sound restart each time the same key is pressed. (Hint use console.dir to take a good look at the audio element)

  3. Style it.

    Add the playing class to a button when it's key is pressed. Then remove it after the appropriate length of time. (Hint: check out the list of possible events.)
