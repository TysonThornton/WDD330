Chp 3 Arrays, Logic, and Loops

  • Create array
    • Const myArray = [];
    • Or
    • Const myArray = new Array();
  • Add values to arrays
    • Assign Superman to element 0
      • heroes[0] = 'Superman';
    • Not one by one
      • const avengers = ['Captain America', 'Iron Man', 'Thor', 'Hulk'];
      • << ['Captain America', 'Iron Man', 'Thor', 'Hulk']
  • Delete value from array
    • delete avengers[3];
    • << true
  • Array Properties
    • Find the length of array called avengers
      • length;
  • Merge arrays with concat()
    • avengers = avengers.concat(['Hulk','Hawkeye', 'Black Widow']);
  • join() method
    • turns the array into a string separated by commas
    • join();
    • << 'Captain America, Iron Man, Thor, Hulk, Hawkeye, Black Widow'
  • Reverse the array – reverse()
    • reverse();
  • Sort the array – sort()
    • Sorting is alphabetical by default for string objects
    • sort();
    • << ['Black Widow', 'Captain America', 'Hawkeye', 'Iron Man', 'Quicksilver', 'ScarletWitch']
  • Search for a value in an array – indexOf()
    • indexOf('Iron Man');
    • << 3
    • OR use includes()
      • includes('Iron Man');
      • << true
      • includes('Thor');
      • << false
  • Sets
    • A collection of unique values – no duplicate values
    • Create
      • const list = new Set();
    • Add values to a set
      • add(2).add(3).add(4);
      • << Set { 1, 2, 3, 4 }
      • OR const numbers = new Set([1,2,3]);
  • Maps
    • Keeps a list of key and value pairs
    • Create maps
      • const romanNumerals = new Map();
    • Add entries
      • set(2,'II').set(3,'III').set(4,'IV').set(5,'V');
  • Logic
    • If statements
      • if (condition) {
      • // code to run if condition is true
    • }else statements
      • if (condition) {
      • // code to run if condition is true
      • } else {
      • // code to run if condition is false
      • }
    • Switch Statements
      • switch (number) {
        • case 4:
        • log('You rolled a four');
        • break;
        • case 5:
        • log('You rolled a five');
        • break;
        • case 6:
        • log('You rolled a six');
        • break;
        • default:
        • log('You rolled a number less than four');
        • break;
        • }
        • Is the same as:
      • if (number === 4) {
      • log('You rolled a four');
      • } else if (number === 5) {
      • log('You rolled a five');
      • } else if(number === 6){
      • log('You rolled a six');
      • } else {
      • log('You rolled a number less than four');
      • }
    • While loops
      • while (condition) {
      • // do something
      • }

 

  • Do While loops
    • do {
    • do something
    • } while(condition)
  • For Loops
    • for (initialization ; condition ; after) { do something }
