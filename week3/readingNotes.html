Chp 5: Objects

  • Arrays and Functions are built in Objects. There can also be user defined objects.
  • Object Literal
    • Javascript Object: self-contained set of related values and functions
    • Its like a database of values.
    • An object is created in Javascript by wrapping all its properties and methods in curly braces.
    • Example

const superman = {

    name: 'Superman',

    'real name': 'Clark Kent',

    height: 75,

    weight: 235,

    hero: true,

    villain: false,

    allies: ['Batman','Supergirl','Superboy'],

    fly() {

        return 'Up, up and away!';

    }

};

  • Call method
    • fly()
      • << 'Up, up and away!'
    • superman['fly']()
      • << 'Up, up and away!'
    • Loop through an object’s properties and methods
      • USE FOR IN LOOP
        • for(const key in superman) {

    console.log(key + ": " + superman[key]);

}

<< "name: Superman"

<< "real name: Clark Kent"

<< "height: 75"

<< "weight: 235"

<< "hero: true"

<< "villain: false"

<< "allies: Batman,Supergirl,Superboy"

<< "fly: function (){

  console.log(\"Up, up and away!\");

}"

  • This
    • Refers to the object that it is within. Used to gain access to the object’s properties
    • Example

const dice = {

    sides: 6,

    roll() {

        return Math.floor(this.sides * Math.random()) + 1;

    }

}

  • JSON
    • Looks like an object but the property names are double quoted
  • Math object
    • Built in object that can do a ton of mathematical operations
    • Math Methods
      • Absolute values: Math.abs()
      • Rounding
        • ceil()
        • floor()
        • round()
        • trunc()
      • Powers and Roots
        • exp()
        • pow()
        • sqrt()
        • cbrt()
        • hpot()

Chp 6 The Document Object Model

  • DOM allows access to elements of a web page and enable interaction with the page by adding and removing elements, changing the order, content and attributes of elements, and can also alter how they are styled
  • DOM treats everything on a website as a node which is just an HTML element
  • Getting Elements
    • ID
      • const h1 = document.getElementById('title');
    • Tag name
      • const listItems = document.getElementsByTagName('li');
    • Class name
      • const heroes = document.getElementsByClassName('hero');
